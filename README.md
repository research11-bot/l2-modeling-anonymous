# l2-modeling-anonymous
Anonymous repository for L2 preposition modeling experiments, including model evaluation, human-like error analysis, and perplexity comparisons across learner corpora.
This repository contains code and data for L2 preposition modeling experiments.

It includes:
- Model evaluation on masked preposition tasks
- Human-like error analysis
- Perplexity evaluation across learner corpora

All experiments are anonymized for review.
